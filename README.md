# StickerSmash

StickerSmash is a delightful and fun mobile application built with React Native and Expo that lets you interact with, edit, and apply stickers to your photos.

## Features
- Choose from a wide variety of stickers.
- Apply, rotate, and scale stickers onto your favorite images.
- Save and share your sticker-smashed photos with friends.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the application:
   ```bash
   npx expo start
   ```
